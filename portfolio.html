June 2021

I started by designing my own map ADT. Using data compiled by the CSE 331 Software Design and Implementation staff, I used my map ADT and Dijkstra's algorithm to plot the shortest paths across the UW campus. The front-end was developed with React and JS, while the back-end was developed with a Java Spark server.

March 2021

Using census data from 2010, I worked to overlay a US map with a grid and query the grid for a population within a range. This was done while exploring parallel programming and concurrency using the fork-join framework.

December 2020

In a 24 hour coding marathon that led to a 1st place prize, I developed the front-end look and feel of our poetry website. The site allows a user to post a poem alongside other related poems via tags. The interface allows a user to review material based on similar tags or completely randomly. I used the bootstrap framework to organize the site.
Text overlay code used in part from w3schools HowTo .
